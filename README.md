# 💰 Asisten Keuangan Mahasiswa (CLI)

Program **Asisten Keuangan Mahasiswa** adalah aplikasi berbasis **Command Line Interface (CLI)** yang dibuat menggunakan bahasa **C** untuk membantu mahasiswa dalam mengelola keuangan sehari-hari secara sederhana dan praktis.

## 📌 Deskripsi

Program ini dirancang untuk mempermudah perhitungan keuangan dasar yang sering dialami mahasiswa, seperti menghitung diskon, membagi tagihan, menghitung bunga tabungan, dan mengatur uang jajan harian.

## 🚀 Fitur Utama

Program ini memiliki 4 fitur utama:

### 1. 🛒 Kalkulator Diskon Belanja

Menghitung harga akhir setelah diskon.

* Input: harga awal barang, persentase diskon
* Output: harga akhir yang harus dibayar

### 2. 🍽️ Kalkulator Split Bill

Membagi total tagihan ke beberapa orang.

* Input: total tagihan, jumlah orang
* Output: jumlah yang harus dibayar per orang

### 3. 💵 Kalkulator Bunga Tabungan

Menghitung saldo setelah ditambah bunga bulanan.

* Input: saldo awal, persentase bunga
* Output: total saldo setelah bunga

### 4. 📊 Kalkulator Sisa Uang Jajan Harian

Menghitung sisa uang setelah pengeluaran.

* Input: uang saku harian, estimasi pengeluaran
* Output: sisa uang

## 🛠️ Teknologi

* Bahasa Pemrograman: **C**
* Interface: **CLI (Command Line Interface)**
